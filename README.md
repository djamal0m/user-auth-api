## Installation

Install the dependencies before running the server:

```bash
npm install
# or
yarn install
#or
pnpm install
# or
bun install
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Server runs on localhost port 8081 ("http://127.0.0.1:8081")
